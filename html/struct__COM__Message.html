Detailed Description

A struct that holds the contents of a COM message.

To read or write a byte to the I/O port COM1, a process sends a message to the COM process. The contents of that message are stored in a COM_Message struct. The members of this struct include the following:

char* output_buffer string to be output to the COM1 port
char* input_buffer buffer where input from the COM1 port will be stored
int len_input_buffer the number of bytes to be read from the COM1 port
