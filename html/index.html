TOS API Documentation

Welcome to the documentation for the Train Operating System for CSC720.
Throughout the semester, you will be implementing the core features of a functional operating system for a model train!
While doing so, you will get hands-on experience with many important concepts of modern operating systems, including among others:

  • Processes
  • Preemptive multitasking
  • Task scheduling
  • Context switches
  • Inter-process communication
  • Interrupts
  • Device drivers

Functions You Will Implement

Throughout the course you will implement the core functions of TOS in the *.c files located in the tos/kernel directory.
You will be able to test your implementations using the TOS test center, explained in the lecture slides.

Features Provided For You

Everything else is provided for you, and it is recommended that you look around all the files to become familiar with the features and overall organization of TOS.
Some important features that you will use extensively later in the course include:
(1) a serial line communication simulator, with which you will practice I/O through the COM1 port.
(2) a train simulator, which you will use to develop and test your final project.
