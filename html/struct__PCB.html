_PCB Struct Reference

#include <kernel.h>

Detailed Description

Struct that holds the information associated with a TOS process.

This struct is used to store information about a TOS process. Each entry in the pcb[] array is one of these structs. The members of this struct are as follows:

unsigned magic A "magic" value used to verify that the record containing this value is indeed a PCB entry. The value is given by MAGIC_PCB.
unsigned used An integer value indicating whether this PCB entry in the pcb[] array has been assigned to a process yet. Once assigned, this PCB entry is never freed up or re-assigned to a different process.
unsigned short priority An integer from 0 to 7 corresponding to the priority leve of this process.
unsigned short state An integer associated by "define" directives to STATE_READY, STATE_SEND_BLOCKED, STATE_RECEIVE_BLOCKED, STATE_MESSAGE_BLOCKED, STATE_REPLY_BLOCKED, STATE_INTR_BLOCKED, OR STATE_ZOMBIE.
MEM_ADDR esp The value of the stack pointer for this process.
PROCESS param_proc A temporary storage location for the PROCESS that was passed as an input to send() or message().
void* param_data A temporary storage location for the data that was passed as an input to send() or message().
PORT first_port The first PORT in a singly-linked list of PORTs owned by this process.
PROCESS next_blocked The next PROCESS in the send-blocked list of processes.
PROCESS next, PROCESS prev The next and previous processes, used for maintaining the doubly-linked list at each priority level of the ready_queue.
char* name The name of this process, as a character array.

The array pcb[] holds 20 of these PCB structs. The typedef PCB is created to serve as another name for this struct. The underscore in the name _PCB indicates that it will not be necessary to refer directly to the PCB struct when implementing the TOS API.
