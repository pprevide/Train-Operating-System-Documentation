#include <kernel.h>

Detailed Description

Holds information associated with a TOS process's port, used for inter-process communication.

Each process in TOS owns one or more ports. While each process can own multiple ports, a port can only be owned by exactly one process. The maximum number of ports across all TOS processes is defined as MAX_PORTS (i.e., MAX_PROCS*2, equal to 40).

Note that PORT is a typedef for "struct _PORT_DEF*", where the leading underscores in the struct's name indicate that the typedef PORT should be used when implementing TOS. (The "_PORT_DEF" link in the table below and throughout this documentation will redirect to this page.)

The members of this struct are described below:

unsigned magic A "magic" value used to verify that the record being accessed is indeed a PORT entry. The value is given by MAGIC_PORT.
unsigned used An unsigned integer indicating whether the entry in the array of ports is being used or not. The array of ports is initialized by init_ipc().
unsigned open An unsigned integer indicating whether the port is open or closed.
PROCESS owner The process which owns this port. A port can have exactly one owner.
PROCESS blocked_list_head The first process on the send-blocked list of this port.
PROCESS blocked_list_tail The last process on the send-blocked list of this port. The tail of the list is maintained in order to efficiently add new processes to the end of the list.
struct _PORT_DEF *next The next port (if any) owned by the owner of this port.
