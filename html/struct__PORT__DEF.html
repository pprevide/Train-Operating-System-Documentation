#include <kernel.h>

Detailed Description

A data structure that holds the members associated with a TOS process's port, used for inter-process communication.

The underscore in the struct's name indicates that it will not be necessary to refer directly by name to this data structure when implementing TOS. Rather, the typedef PORT can be used to access this struct more conveniently. The members of this struct are described below:

unsigned magic A "magic" value used to verify that the record being accessed is indeed a PORT entry. The value is given by MAGIC_PORT.
unsigned used An unsigned integer indicating whether the port slot in the port[] array is being used or not.
unsigned open An unsigned integer indicating whether the port is open or closed.
PROCESS owner The process which owns this port. A PORT can have exactly one owner.
PROCESS blocked_list_head The first process on the send-blocked list of this port.
PROCESS blocked_list_tail The last process on the send-blocked list of this port. The tail of the list is maintained in order to efficiently add new processes to the end of the list.
struct _PORT_DEF *next The next port (if any) owned by the owner of this port.
